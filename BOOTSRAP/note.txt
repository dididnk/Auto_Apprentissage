## BOOTSTRAP NOTES
==================

Taille de l'écran (12 espaces)
------------------
xs <    768px : phones
sm >=   768px : tablettes
md >=   992px : desktop
lg >=   1200px: large desktop

Taille de container
------------------
container-fluid : toute la largeur
container       : 90% de la largeur
-en xs  : tout l'écran
-en sm  : 750px +- 90% de l'écran
-en md  : IDEM
-en lg  : IDEM
